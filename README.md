# Calgary Housing Market Analysis

## Project Overview

This project analyzes Calgary's residential housing market using real-world data.

The goal is to identify housing market trends, community performance, affordability, and investment opportunities through data analysis.

---

## Project Objectives

- Analyze Calgary housing price trends
- Compare different communities
- Explore housing affordability
- Build interactive Power BI dashboards
- Provide business insights and recommendations
